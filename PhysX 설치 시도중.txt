그리고 나서 비쥬얼스튜디오에서 x64로 하고있는지
MD MT 설정이 일치하는지 확인해야함 

X64 확인은, 위에 실행버튼 옆에 x64적혀있음
MD MT는,
프로젝트 속성 -> 코드 생성 -> C/ C++  -> 런 타임라이브러리 이거임
내 프로젝트 기준으로는
Release = /MD
Debug = /MDd
로 작동하고 있음

나중에 링크할때, MD랑 MDd랑 걸면 터진다고 하니까, 나중에 확인 잘 해야할듯? << 더 찾아볼 필요가 있음
